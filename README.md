# Election_data_analytics
End-to-end SQL project involving database design, data querying, and analytical insights on election data, including constituency trends, party performance, and voter behavior.
Election Data Analysis using SQL

## 📊 Project Overview
This project analyzes election data across constituencies using SQL.

## 🗂️ Database Tables
- Constituency
- Candidates
- Party
- Results

## 🔍 Key Analysis
- Voter distribution across constituencies
- Candidate participation analysis
- Party-wise vote share
- Winning candidates and margins
- Voter turnout percentage

## 🛠️ Tools Used
- MySQL

## 📈 Key Insights
- Identified high and low voter constituencies
- Analyzed party dominance
- Evaluated election competitiveness using winning margins
- Measured voter turnout

## 📁 Files
- DDL_AND_DML.sql → Database creation
- SELECT_QUERIES.sql → Basic analysis
- JOINS.sql → Advanced analysis
